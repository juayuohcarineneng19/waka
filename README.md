
# Project Name: WAKA 
This project is aim at showing the beautiful beaches and resorts in Cameroon that i have visited. 
## Built With

- React.js
- React Router
- React Hooks
- React Context API
- Styled Components

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repo
https://github.com/juayuohcarineneng19/waka.git
2. Install NPM packages
npm install
```
3. Start local dev server 
```sh
npm start
```
